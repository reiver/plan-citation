# plan-citation

**plan-citation** creates a new `.citation` file based on the `.plan` format.

**plan-citation** sets the `uuid` and `unix-time` to the current unix-time stamp.

## Usage

To write a `.citation` file to the current directory:

```
plan-citation
```

To write to stdout instead of a file:

```
plan-citation --stdout
```

To write a `.citation` file to a specific directory:

```
plan-citation /path/to/directory
```
